Để chạy được phần mềm đầu tiên cần sử dụng terminal di chuyển đến thư mục dự án

hãy đảm bảo là trên máy đã có tải node
sử dụng lệnh 'npm install' để tải về các package cần thiết cho dự án

sao khi tải package
sử dụng lệnh 'npm start' để chạy dự án

dự án được chạy ở localhost:3000
